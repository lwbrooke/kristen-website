# lbrooke.com

My personal website

## Technologies

- javascript, html, css
- angularjs
- angular-material
- webpack
- nodejs, npm

## Install and run development server

- git clone https://github.com/lwbrooke/lbrooke.com
- cd lbrooke.com
- npm install
- npm run server
- navigate to http://localhost:8080

Seeded from [preboot/angular-webpack](https://github.com/preboot/angular-webpack)
